*Breakout Room - How to make a generative NFT collection*

Starting from CryptoPunks to that of Bored Ape Yacht Club, NFTs have just begun taking hold of the blockchain industry making big moves and have since emerged as the next biggest thing next to the cryptocurrencies. With celebrity endorsements and jaw-dropping sales from several high-profile NFTs, interest in making NFTs art has never really been that high.

NFT is an acronym for Non Fungible Token. It is a non-interchangeable unit of data stored on a blockchain in the form of a digital ledger. These data units act as proof of ownership for objects associated with them that can be created, traded and owned. The objects can be real or virtual, but are typically unique and scarce. Unlike cryptocurrencies, each token is unique and cannot be exchanged. Imagine that a U.S. dollar is synonymous with a Bitcoin and an NFT with the painting of Mona Lisa. Each dollar bill can be exchanged for another dollar bill of the same value, but there can only be one Mona Lisa painting. Most NFTs are large collections of similar themes, algorithmically generated. There are many algorithmic approaches to generating NFTs, in the breakout room we had a presentation and walkthrough of one of the methods made popular by the Weirdwhales collection using a simple Python script. Most generative NFT arts share a basic set of attributes or properties that can be arbitrarily layered to create a unique work of art. Take CryptoPunks for example, there are 5 basic punk types (alien, monkey, zombie, etc.) and 87 possible attributions (beanie, cigarette, earring, etc.). The attributes can also be modified programmatically to emphasize the rarity of certain figures. So in order to create a generative NFT collection, all you need is a little bit of Graphic Design (just to make a few objects), and you'll need a little bit of programming - however, this last part is extremely easy to do nowadays. That has to do with the fact that a lot of the resources that are needed have been released as Open Source projects on the web. This makes it incredibly easy to create your own generative collection of let's say 10.000 NFT's. All generated in a few minutes by a simple Python script and a few graphic models, either in PNG/JPG or even SVG format. It's even possible to make 3D objects by creating the images through a 3D software like Blender for example.

What I also learned from the breakout room session what that nowadays, because it's so simple to create a full generative NFT collection and upload it on the markets for free, it is not the one-way ticket to success. This has to do with the fact that too many people are using this trick already. This means that it's not so much about the technique behind it anymore, but about the marketing aspect that comes along with it. This is the part that is being forgotten most of the time. It's easy to create an NFT collection, but to sell it is a whole different story.
